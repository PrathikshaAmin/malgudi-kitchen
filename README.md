# Malgudi Kitchen - Restaurant Landing Page

A simple restaurant landing page built using HTML and CSS.

## Sections
- Hero
- Popular Dishes
- About Restaurant
- Testimonials
- Contact
- Footer

## Features
- Responsive design (cards adjust for mobile screens)
- Hover animations on buttons and cards
- Background image
- Google Fonts (Poppins & Playfair Display)
- CSS Variables for colors

## Files
- `index.html` - page structure
- `style.css` - styling

## How to run
Open `index.html` in any web browser.
